CNMP Buffet Web App
=========================

-Web App donde se muestra el menu del buffet del Club Nautico Mar del Plata, donde los comensales pueden ver los platos divididos por categorias y junto a cada una de las comidas, se presenta un boton clickeable indicando el precio del plato. Al final de la pagina se muestra el subtotal de los platos a√±adidos al carrito de compras.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Utilice para el desarrollo
- HTML
- CSS
- Javascript
- Node.js
- Express
- SQL Server
